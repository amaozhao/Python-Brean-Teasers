# Raymond Hettinger 的序言

在我的 Python 会议演讲中，我经常询问听众，“你学到了什么新东西吗？”一次又一次地得到一个“是”让每个人都感到高兴，并告诉我们我们的时间花得很好。 Miki 的脑筋急转弯系列将为您带来即时的满足感，每个谜题一次。期待他的“啊哈！”流带来很多乐趣时刻。

Miki 和我一起工作了三次：一次是在贸易公司，一次是在网络服务公司，另一次是作为 Python 培训师。和他一起工作总是会给你“我学到了新东西”的体验。

作为培训师，我们发现一项关键技能是能够阅读代码并真正了解代码的作用。通过 Miki 精心挑选的示例，您可以快速学习这项基本技能。他给了你一个有趣的代码片段，让你做一个预测，然后温柔地解释结果。作为锦上添花，他还提供了权威参考文献的链接，以加深您的知识。

Python 不是一门难学的语言，但它的功能远不止眼前一亮。当你真的不知道这门语言时，很容易假设你很了解这门语言。 Dunning-Kruger 效应在 Python 世界中无处不在。 Miki 的脑筋急转弯将帮助您快速发现您不知道的内容，他的解释将填补缺失的知识以建立您的专业知识。
